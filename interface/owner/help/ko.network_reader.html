전역 단축키 일람

  • Q: 블로그 화면으로 이동
  • R: RSS 이웃글 보기로 이동
  • A: 리스트에서 앞페이지로 이동
  • S: 리스트에서 뒷페이지로 이동
  • ?: 이 도움말 보기

리더 단축키 일람

  • A,H : 리더의 앞 글을 읽습니다.
  • S,L : 리더의 다음 글을 읽습니다.
  • D : 현재 글을 새 창에서 엽니다.
  • F : 읽지 않은 글만 봅니다.
  • G : 별표를 매긴 글만 봅니다.
  • T : 모든 글을 새로 읽습니다.
  • W : 현재 글에 별표를 주거나 지웁니다.
  • J : 화면을 위로 스크롤합니다.
  • K : 화면을 아래로 스크롤합니다.
